C++ is better than Java. you want to go to a supermarket, your sedan is better. If you want to go to another country on the other side of the ocean, jumbo is better. The same thing happens, more or less, between C++ and Java. C++ is a jumbo jet and Java is a beautiful sedan. Which is better? Therefore, for an intermediate program on a cell phone, Java is better. For a very intense CPU program, on the same cell phone, or even more difficult, on a main computer, C++ is definitely better.
